# PWA Icons

This directory contains PWA icons for AurisVoice.

## Required Files

The following PNG files should be created from the SVG placeholders:

- `icon-192x192.png` (192x192 pixels)
- `icon-192x192-maskable.png` (192x192 pixels, maskable)
- `icon-512x512.png` (512x512 pixels)
- `icon-512x512-maskable.png` (512x512 pixels, maskable)

## Generating PNGs

You can use tools like:
- ImageMagick: `convert icon-192x192.svg icon-192x192.png`
- Online converters
- Design tools (Figma, Sketch, etc.)

For maskable icons, ensure they have safe zones (80% of the icon should be visible when masked).
