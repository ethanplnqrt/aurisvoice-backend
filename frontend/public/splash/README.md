# iOS Splash Screens

This directory contains splash screen images for iOS PWA installation.

## Required Files (Placeholders)

The following PNG files should be created:

- `iphone-se.png` (640x1136) - iPhone SE
- `iphone-8.png` (750x1334) - iPhone 8/7/6s
- `iphone-x.png` (1125x2436) - iPhone X/XS
- `iphone-11-pro.png` (1242x2688) - iPhone 11 Pro Max
- `iphone-12-pro-max.png` (1242x2688) - iPhone 12 Pro Max / 13 Pro Max
- `ipad.png` (1536x2048) - iPad

## Design Guidelines

- Background color: `#020617` (matches manifest background_color)
- Center logo/icon: AurisVoice logo centered
- Text: Optional "AurisVoice" text below logo
- Safe area: Keep important content within safe area (notch area)

## Generating Splash Screens

You can use:
- Design tools (Figma, Sketch, Adobe XD)
- Online generators: https://appsco.pe/developer/splash-screens
- ImageMagick scripts
